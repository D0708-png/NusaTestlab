# NusaTestLab Overview

NusaTestLab adalah standalone SaaS testing tool.

Aplikasi ini dibuat untuk membantu developer melakukan pengujian terhadap sistem SaaS, terutama sistem yang memiliki:

- multi-tenant architecture
- role-based access control
- API-based backend
- business logic kompleks
- laporan bisnis
- fitur AI atau automation

## Tujuan Awal

Tujuan awal NusaTestLab adalah menjadi alat testing untuk sistem seperti AI UMKM.

Namun, arsitekturnya dibuat generic agar bisa digunakan untuk SaaS lain di masa depan.

## Target v0.1.0

Fokus versi pertama:

- membuat core project structure
- membuat target SaaS configuration
- membuat profile system
- membuat AI UMKM testing profile
- membuat inventory testing scenario
- membuat report validation
- membuat basic security testing
- membuat hasil report dalam JSON dan Markdown
