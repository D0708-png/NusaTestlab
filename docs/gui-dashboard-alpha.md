# GUI Dashboard Alpha

NusaTestLab GUI Dashboard Alpha introduces a local static dashboard for reading latest test reports generated by the CLI.

## Current Scope

This is not a hosted SaaS dashboard yet. It is a local HTML dashboard generated from existing NusaTestLab report files.

The dashboard can summarize:

- latest suite report
- latest browser report
- latest browser scenario report
- risk score and status
- report file availability
- quick command hints

## Commands

Build dashboard:

```bash
npm run gui:build
```

Open dashboard:

```bash
npm run gui:open
```

Direct CLI usage:

```bash
npx tsx src/index.ts gui:build
npx tsx src/index.ts gui:open
```

## Output

The generated dashboard is stored at:

```txt
results/gui-dashboard/index.html
```

Open this file in your browser to view the dashboard.

## Recommended Flow

Run some tests first:

```bash
npm run suites:ai-umkm
npm run browser:example
npm run browser:scenario:example
npm run gui:build
```

Then open:

```txt
results/gui-dashboard/index.html
```

## Current Limitation

This alpha dashboard is read-only.

It does not yet provide:

- scenario creation from GUI
- run test from GUI button
- login/workspace management
- hosted database
- live worker queue

Those belong to the next GUI phases.
